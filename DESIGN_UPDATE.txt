📊 FITUR DASHBOARD
═══════════════════════════════════════════════════════════════════

1. Total Mahasiswa
   - Menghitung semua mahasiswa di database
   - Display: Angka besar di card biru

2. Total Jurusan
   - Menghitung distinct program_studi
   - Display: Angka besar di card hijau

3. Status Sistem
   - Static: "Aktif"
   - Display: Text besar di card kuning

4. Mahasiswa per Program Studi
   - Query: GROUP BY program_studi
   - Display: Grid 2 kolom
   - Format: Nama jurusan (capitalized) + jumlah

5. Mahasiswa Terbaru
   - Query: ORDER BY created_at DESC LIMIT 5
   - Display: Tabel 2 kolom (Nama, Jurusan)

- Type: Dropdown select

Empty State:
- Menampilkan pesan khusus saat filter aktif tapi kosong
