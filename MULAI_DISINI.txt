🚀 CARA MENJALANKAN LOKAL (5 MENIT)
═══════════════════════════════════════════════════════════════════

Buka terminal/command prompt, lalu jalankan:

1️⃣ Install dependencies:
   composer install
   npm install

2️⃣ Setup database:
   php artisan migrate:fresh --seed

3️⃣ Build assets:
   npm run build

4️⃣ Jalankan server:
   php artisan serve

5️⃣ Buka browser:
   http://127.0.0.1:8000

6️⃣ Login:
   Email: admin@gmail.com
   Password: admin123

7️⃣ Test fitur:
   Klik menu "Mahasiswa" → Test Create, Edit, Delete

❓ TROUBLESHOOTING CEPAT
═══════════════════════════════════════════════════════════════════

❌ composer: command not found
   → Install Composer: https://getcomposer.org

❌ npm: command not found
   → Install Node.js: https://nodejs.org

❌ Error database
   → Pastikan sudah run: php artisan migrate:fresh --seed

❌ CSS tidak muncul
   → Jalankan: npm run build

❌ Error 500
   → Cek file .env sudah ada
   → Jalankan: php artisan key:generate
